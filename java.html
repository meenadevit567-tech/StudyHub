1. Introduction to Java

Java is an object-oriented, platform-independent programming language widely used for backend systems, Android apps, and enterprise applications.

  • Write once, run anywhere
  • Secure and robust
  • Object-oriented

2. Variables and Data Types

int a = 10;
float price = 25.5f;
char grade = 'A';
boolean isValid = true;
String name = "Meena";

3. Conditional Statements

int num = 10;

if(num % 2 == 0){
    System.out.println("Even");
}else{
    System.out.println("Odd");
}

4. Loops

for(int i=1;i<=5;i++){
    System.out.println(i);
}

int i=1;
while(i<=5){
    System.out.println(i);
    i++;
}

5. Methods

static int add(int a,int b){
    return a+b;
}

public static void main(String[] args){
    System.out.println(add(3,5));
}

6. OOP Concepts

class Student{
    void display(){
        System.out.println("Hello Student");
    }
}

class Main{
    public static void main(String[] args){
        Student s = new Student();
        s.display();
    }
}

7. Exception Handling

try{
    int x = 10/0;
}
catch(Exception e){
    System.out.println("Error occurred");
}

8. Collections (ArrayList)

import java.util.*;

ArrayList list = new ArrayList<>();
list.add(10);
list.add(20);
System.out.println(list);
