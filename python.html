🐍 Python Study Material

1. Introduction to Python

Python is a high-level, interpreted programming language that is easy to learn and simple to use. It is widely used in backend development, data science, automation, and web applications.

Features of Python:

  • Simple and readable syntax
  • Platform independent
  • Large standard library
  • Supports object-oriented programming
---

2. Variables and Data Types

Variables are used to store values. Python does not require data type declaration. 

x = 10
name = "Meena"
price = 25.5
isActive = True

Common Data Types:

  • int – whole numbers
  • float – decimal numbers
  • str – text
  • bool – True / False
---

3. Conditional Statements

Conditional statements are used to execute code based on conditions. 

num = 10

if num > 0:
    print("Positive number")
else:
    print("Negative number")
---

4. Loops (for and while)

Loops are used to repeat a block of code.

For Loop:

for i in range(1, 6):
    print(i)

While Loop:

i = 1
while i <= 5:
    print(i)
    i += 1
---

5. Functions

Functions are reusable blocks of code. 

def add(a, b):
    return a + b

result = add(5, 3)
print(result)
---

6. Lists, Tuples, Sets, Dictionaries

List:

nums = [1, 2, 3, 4]

Tuple:

data = (10, 20)

Set:

s = {1, 2, 3}

Dictionary:

student = {
  "name": "Meena",
  "age": 20
}
---

7. Object-Oriented Programming (OOP)

Python supports OOP concepts like class and object. 

class Student:
    def display(self):
        print("Welcome Student")

obj = Student()
obj.display()
---

8. File Handling

File handling is used to store and read data from files.

Write to a file:

file = open("data.txt", "w")
file.write("Hello Python")
file.close()

Read from a file:

file = open("data.txt", "r")
print(file.read())
file.close()
